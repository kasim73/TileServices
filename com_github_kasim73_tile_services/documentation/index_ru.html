Карты из Интернета

 

 

Описание

Модуль Карты из Интернета предназначен для открытия карт с картографических сайтов (сервисов) сети Интернет.

Используя Карты из Интернета, можно:

  • открывать карты в виде слоя;
  • сохранять карты в виде TAB-файла на своём компьютере. В TAB-файл записывается только XML-ссылка на карту, что позволяет позднее её открывать напрямую из ГИС Аксиома, без модуля Карты из Интернета.

Для работы с модулем Карты из Интернета, а также для открытия карт, сохранённых с помощью этого модуля, вам потребуется работающее интернет-соединение.

Установка модуля

Модуль находится в репозитории ГИС Аксиома. Для установки модуля на вкладке Основные выберите команду Модули. В диалоге выберите вкладку Дополнительные модули, загрузите список доступных модулей и установите модуль Карты из Интернета.

После загрузки модуля на вкладке Основные появится кнопка Карты из Интернета, с помощью которой открывается панель каталога карт. Кнопка имеет два фиксированных положения:

  • утопленное − в данном положении панель отображается;
  • отжатое − панель скрыта.

При необходимости вы можете отключить или удалить модуль из вкладки Установлено диалога Модули.

Каталог карт

Каталог карт, размещённый на панели, содержит список, элементами которого являются:

  • сервисы − названия сервисов, которые могут предоставлять карты;
  • карты − список карт,  схем и спутниковых снимков, которые можно просматривать с данного сервиса.

Кнопка Открыть

Открывает и загружает карту, выбранную в списке Карты из Интернета, в новом или активном окне Карты.

Кнопка Сохранить

Сохраняет ссылку на интернет-карту в TAB-файле. В TAB-файл записывается только XML-ссылка на сервис, картографическая информация не переносится. Позднее этот TAB-файл можно открыть в ГИС Аксиома, не используя модуль Карты из Интернета. Для открытия полученного TAB-файла требуется работающее соединение с Интернетом.

Кнопка Развернуть все / Свернуть все

Разворачивает или сворачивает весь список карт, перечисленных в каталоге карт.

Кнопка Обновить

Полностью обновляет каталог карт с Web-сайта модуля. Сервисы могут меняться. Если та или иная карта более недоступна, сообщите нам, и мы внесём изменения.

Замечания о правообладателях

У каждой карты есть правообладатель. Вы можете пользоваться картой в соответствии с выбранной для неё лицензией. Ссылка на лицензию отображается в информации о карте под списком сервисов.

Если у вас есть предложения по добавлению нового сервиса, пожалуйста, сообщите нам.
